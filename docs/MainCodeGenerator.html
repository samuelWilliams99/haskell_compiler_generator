MainCodeGenerator

Description

The code generated by this file handles all the IO actions for the generate compiler, and acts as the entry point. This code will differ greatly depending on whether or not includes are supported, since the majority of the handling for this feature is within this file. The IO actions supported by this file include argument and flag handling for command-line use, input and output file path specification, and verbose command-line output.
